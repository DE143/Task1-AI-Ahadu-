# 💰 Expense Tracker Pro

A professional expense tracking application built with Python and Tkinter.

## Features

- ✅ Track income and expenses
- ✅ Categorize transactions
- ✅ Calculate total balance automatically
- ✅ Visual spending trends with bar charts
- ✅ Persistent data storage (JSON)
- ✅ Modern, professional UI
- ✅ Real-time summary cards

## Requirements

```bash
pip install matplotlib
```

## How to Run

```bash
python expense_gui.py
```

## Usage

1. **Add Transaction**: Select type (Income/Expense), enter amount, choose category, and click "Add Transaction"
2. **View Summary**: See your total balance, income, and expenses at a glance
3. **Analyze Spending**: Check the bar chart to see spending by category
4. **Review History**: View all recent transactions in the list
5. **Clear Data**: Use "Clear All" button to reset all transactions

## Categories

- Food
- Transport
- Shopping
- Bills
- Entertainment
- Healthcare
- Salary
- Other

## Data Storage

All transactions are automatically saved to `expense_data.json` in the same directory.
